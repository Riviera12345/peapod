char* args_t::pidfile

A C string containing the path to the PID file.

Providing -d means that this will be set to something, because running as a daemon requires a PID file. May be the argument to -p. Defaults to PEAPOD_PID_PATH.

char* args_t::conffile

A C string containing the path to the config file.

May be the argument to -c. Defaults to PEAPOD_CONF_PATH.

uint8_t args_t::level

Logging level.

Defaults to LOG_NOTICE (5). Providing -v increments this up to LOG_DEBUGPKT (8).

See also
log.h
char* args_t::logfile

The path to the log file.

Independently of whether logs are emitted to the console and/or syslog, controls whether logs are emitted to a log file. If -l is not provided, remains NULL, and a log file is not used. Otherwise, may be the optional argument to -l, or the default of PEAPOD_LOG_PATH.
